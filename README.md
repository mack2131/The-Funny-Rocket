# The-Funny-Rocket
![Alt text](img/1.jpg?raw=true "Title")
Это простой протоитп запуска ракеты из одной точки земного шара в другую.
По умолчанию, база, откуда летит ракета, находится на южном полюсе.
Для запуска ракеты необходимо ввести команду
attack moscow def
и нажать ввод
Кроме moscow можно ввести new-yor
def - это тип ракеты по умолчанию
другиъ ракет и городов нет
можно крутить-приближать и отдалаять планету
